description: How to create a new dataset in Envision Portal.
---

## Steps

1. Log in and go to `/app/datasets`.
2. Click **New Dataset**.
3. Provide the initial details:
   - **Title** - The name of the dataset
   - **Description** - A short summary of the dataset
   - **Version** - Starting version (typically `1.0.0`)
   - **Type** - The resource type (e.g., dataset, image collection)
4. Submit the form. You will be redirected to the new dataset's overview page.

From there you can fill in metadata, upload files, and eventually publish.

## Draft Datasets

All new datasets start as drafts. A draft is only visible to team members with access. You can work on a draft as long as needed before publishing.

## Sharing a Draft

You can invite collaborators to a draft dataset before publishing. Go to the **Permissions** tab of the dataset and use the invite form to add team members by email. See [Roles and Permissions](/access/roles-and-permissions) for details on what each role can do.
