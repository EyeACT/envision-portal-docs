The **Discover** page at `/datasets` is publicly accessible and lists all published datasets on the platform. No account is required to browse.

## Finding Datasets

Use the search bar to search by title, description, or keywords. Results update as you type.

You can also filter by:

- **Access type** - Public or controlled
- **Subject / keywords**
- **Resource type**

## Dataset Detail Page

Click on any dataset to open its detail page at `/datasets/[datasetId]`. The detail page shows:

- Title and description
- Contributors and affiliations
- Related identifiers (publications, source datasets)
- Access type and rights information
- Dataset files index
- README content
- Changelog

From this page you can request access to the dataset.

## External Datasets

Envision Portal also indexes datasets from external repositories. These are registered via the API and appear in search results alongside datasets hosted on the platform. External dataset entries link out to the original source.

## Dataset Versions

If a dataset has multiple published versions, you can switch between them using the version selector on the detail page. Each version has its own metadata, files, and README.
